# LockLens Android

Рабочая тестовая версия LockLens.

## Что должно работать

- русский интерфейс;
- живой предпросмотр камеры на главной;
- живой предпросмотр камеры во вкладке «Камера»;
- выбор задней и фронтальной камеры;
- выбор качества 480p / 720p / 1080p / 4K;
- запрос разрешений;
- включение и выключение аудио;
- выбор места сохранения: внутренняя библиотека, галерея, своя папка;
- запись через CameraX;
- Foreground Service;
- уведомление с кнопкой «Стоп»;
- остановка записи из приложения;
- список видео;
- открыть видео;
- поделиться видео;
- удалить видео;
- экспортировать видео в галерею.

## Сборка

GitHub Actions → Build Debug APK → Download artifact `locklens-debug-apk`.

Перед установкой новой debug-сборки удалите старую debug-версию LockLens с телефона.
